Introducción a CSS

CSS, o Cascading Style Sheets (Hojas de Estilo en Cascada), es un lenguaje utilizado para definir y aplicar estilos a documentos HTML. Gracias a CSS se separa el contenido (HTML) de la presentación visual, lo que permite modificar el estilo de un sitio web sin alterar su estructura.

¿Qué son los selectores en CSS?

Los selectores son patrones que se utilizan para seleccionar los elementos HTML a los que se les aplicarán las reglas de estilo. Permiten determinar qué elementos serán afectados y definen la manera en que se aplican los estilos.

Tipos de Selectores

  • Selector por tipo: Selecciona todos los elementos de un mismo tipo, como por ejemplo todos los párrafos (p). Ejemplo: p { color: blue; }
  • Selector de clase: Aplica estilos a todos los elementos que tengan asignada una clase específica, identificada mediante un punto (.clase). Se puede reutilizar en varios elementos. Ejemplo: .destacado { font-weight: bold; }
  • Selector de identificador: Se utiliza para aplicar estilos a un elemento único, identificado mediante la almohadilla (#id). Cada identificador debe ser único en la página. Ejemplo: #principal { margin: 20px; }

Propiedades y Valores en CSS

En CSS, cada regla se compone de una o más propiedades y sus respectivos valores. Una propiedad define qué aspecto se desea modificar (por ejemplo, color, font-size, margin), y el valor asignado indica cómo se aplicará ese cambio.
Ejemplo: color: red; indica que el color del texto será rojo.

La combinación adecuada de selectores, propiedades y valores permite aplicar estilos de forma precisa, haciendo que la presentación del sitio web sea consistente y fácil de mantener.

Nota: El uso de diferentes selectores (por tipo, clase e identificador) facilita la organización y especificidad de los estilos, lo que es fundamental para el diseño y mantenimiento de sitios web modernos.
